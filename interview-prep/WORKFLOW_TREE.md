### How Gates Work

```
EACH DAY:  Drill confidence вүҘ 3/5 вҶ’ move to next day
           Drill confidence < 3/5 вҶ’ repeat day before new content

EACH WEEK: Sunday mock вүҘ 8/10 вҶ’ move to next week
           Sunday mock < 8/10 вҶ’ identify weak days вҶ’ re-do Mon-Tue вҶ’ re-test Wed

BIG GATES:
           Mock 3 (Week 3) вҶ’ Pipeline must be smooth. Most important gate.
           Mock 8 (Week 8) вҶ’ Final check. If not ready вҶ’ delay applications.
```
